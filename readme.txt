Stripe Frontend Registration : Register to wordpress site by accepting payment from user using Stripe.

== Description ==

Stripe Frontend Registration : Allow user to Register to your wordpress blog accepting payment from user using Stripe 
This plugin help you to receive an amount from the front end user.
Plugin will accept user card details and after successful payment user will register to your website.
This Plugin also provide the test mode by setting test Secret Key and test Publishable Key from admin area.

== Installation ==

1. Upload the entire Stripe Frontend Registration folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[stripe_form]` in your templates
